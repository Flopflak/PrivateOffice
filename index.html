Getting Started:

  1. Clone the PrivateOffice repository from GitHub by running the command: git clone https://github.com/Flopflak/PrivateOffice.git
  2. Navigate to the project directory and optionally set up a virtual environment for added isolation.
  3. Launch the application by running the command: python main.py
  4. Open your favorite web browser and visit http://localhost to access PrivateOffice. Log in to your account or create a new one to begin creating, editing, and managing your documents securely.
